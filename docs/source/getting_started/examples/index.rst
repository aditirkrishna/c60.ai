Running the Examples
-------------------

To run the examples, you'll need to have C60.ai installed (see :ref:`installation`).

1. Clone the repository:

   .. code-block:: bash

       git clone https://github.com/aditirkrishna/c60.ai.git
       cd c60.ai/examples

2. Install the required dependencies:

   .. code-block:: bash

       pip install -r requirements.txt

3. Launch Jupyter Lab or Jupyter Notebook:

   .. code-block:: bash

       jupyter lab

   or

   .. code-block:: bash

       jupyter notebook

4. Open and run the example notebooks in your browser.
